Trabalhando com Eventos:

    Principais atributos eventos, clique sobre EVENTOS e veja a documentação:

  • onChange (Dispara no momento em que o elemento é alterado)
  • onClick (Dispara com um clique do mouse no elemento)
  • onLoad (Dispara quando a página termina de carregar)
  • onMouseOver (Dispara quando o ponteiro do mouse se move sobre um elemento)
  • onMouseOut (Dispara quando o ponteiro do mouse se move para fora de um elemento)
  • onSubmit (Dispara quando um formulário é enviado)
